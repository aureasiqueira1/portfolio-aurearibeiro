## ⚙️ Como executar

- Clone o projeto em sua máquina:

```bash
  git@github.com:aureasiqueira1/portfolio-aurearibeiro.git
```

- Abra o projeto:

```bash
  cd portfolio-aurearibeiro
```

- Instale as dependencias (com yarn ou npm):

```bash
  yarn install
```

- Inicie o projeto (com yarn ou npm):

```bash
  yarn start
```
